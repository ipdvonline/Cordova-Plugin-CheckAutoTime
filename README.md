# phonegap-CheckAUTOTIME
Plugin to Check if AutoTime enabled on Android

## install
```
yourAppDir$ phonegap plugin add https://github.com/ipdvonline/CheckAUTOTIME.git
```

## usage

```javascript
CheckAUTOTIME.check(function(){
    //AUTOTIME is enabled!

  },
  function(){
    //AUTOTIME is disabled!

  });
```
